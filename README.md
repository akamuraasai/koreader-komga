# Komga Bulk Downloader (KOReader plugin)

Browse a [Komga](https://komga.org) manga server from your [KOReader](https://koreader.rocks)
e-reader, multi-select chapters, and bulk-download them into per-series folders. Reading
progress syncs back to Komga through KOReader's own sync.

![The Komga home hub](website/docs/assets/screenshots/home.png)

## Install

1. Download `komga.koplugin-vYYYY.MM.DD.zip` from the [Releases page](../../releases) and
   extract the `komga.koplugin/` folder into KOReader's plugins directory (Kobo:
   `.adds/koreader/plugins/`, other devices: `<KOReader data dir>/plugins/`).
2. Restart KOReader.

**Full guide:** see the [documentation site](https://akamuraasai.github.io/koreader-komga/) —
[Get Started](https://akamuraasai.github.io/koreader-komga/get-started/) and
[Usage](https://akamuraasai.github.io/koreader-komga/usage/).
