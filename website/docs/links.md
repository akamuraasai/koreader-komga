# Useful links

- **Komga** — <https://komga.org> — the self-hosted comics/manga server this plugin talks to.
- **KOReader** — <https://koreader.rocks> — the e-reader application this plugin runs inside.
- **comicreader.koplugin** — <https://korcomic.github.io/docs/comicreader.koplugin/intro/> —
  a companion KOReader plugin that turns single-page CBZ documents into a paged comic reader.
- **This plugin** — <https://github.com/akamuraasai/koreader-komga> — source, issues, releases.
